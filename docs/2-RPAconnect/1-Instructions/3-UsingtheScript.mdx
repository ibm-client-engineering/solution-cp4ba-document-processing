---
1. Download the latest RPA .wal file [here](https://github.com/ibm-client-engineering/solution-cp4ba-document-processing/blob/main/assets/rpa-connector/scripts/ADP%20Connect.wal)
2. Open ADP Connext.wal with RPA
3. Run the RPA script
4. Select your input file

The script will be run and ADP will recieve all files in the folder that your input file indicates. Your output folder will populate with the results for each file from ADP as well as a summary file containing all results together.

Example:
:::danger
The current version of the ADP API does not reflect the correct confidence number. This must be fixed with an ADP application update to pull the proper confidence values.
:::
